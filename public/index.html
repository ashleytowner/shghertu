Esperantaj Klavaroj

Ĉi tiu retpaĝo prezentas programojn kaj instrukciojn por tajpi esperantajn literojn per via komputilo aŭ poŝtelefono.

This page presents programs and instructions for typing esperanto letters on your computer or phone.

Linux

Plej multaj Linux-a distribuoj jam havas esperantan klavaron. Vi povas trovi ĝin en la agordoj.

Most linux distributions already have an esperanto keyboard. You can find it in the settings.

Android

Vi devos elŝuti aparte liveritan klavaron, kiu subtenas Esperanton. En Android vi havas du eblojn:

You will need to download a third-party keyboard that supports Esperanto. On Android you have two options:

  • Gboard
  • Swiftkey

iOS / iPad OS

Vi devos elŝuti aparte liveritan klavaron, kiu subtenas Esperanton. En iOS, SwiftKey estas la plej bona eblo.

You will need to download a third-party keyboard that supports Esperanto. On iOS, SwiftKey is the best option.

Chrome OS

Ne extistas bonegaj ebloj per Chrome OS, sed tajpi esperantajn literojn estas ebla.

There isn't a great option for Chrome OS, but typiing esperanto characters is possible.

  • La klavaranĝo "US International Keyboard" povas tajpi esperantajn literojn. Ĉi tiu etato prezentas la klavojn kiujn vi devas uzi por esperantaj literoj.

  • Tajlorita klavaranĝo de Google: Google kreis GitHub-an deponejon por aliaj klavaranĝoj. Ekzistas esperantan klavaranĝon, kiu oni povas instali, sed ĝi estas tre mezbona.

    Custom-made keyboard layout by google: Google created a github repository for other keyboard layouts. There is an esperanto keyboard layout, which you can install, but it is very mediocre.

Aliaj Sistemoj por Skribi

Kiam oni ne povas tajpi esperantajn literojn, ekzistas du aliaj sistemoj por prezenti ilin: La h-sistemo kaj la x-sistemo.

When you are unable to type esperanto letters, there are two other systems for representing them: The h-system and the x-system.

H-sistemo

La Akadaemio rekomendas, ke por aferoj, kiujn oni intencis legigi, oni uzu la h-sistemon.

The Akademio recommends, that for things intended to be read, the h-system should be used.

Esperanta Litero H-Sistemo
ĉ ch
ĝ gh
ĥ hh
ĵ jh
ŝ sh
ŭ u

X-Sistemo

Kvankam la x-sistemo nun estas tre populara ĉie, la Akademio rekomendas, ke oni nur uzas ĝin, kiam oni ne povas uzi la h-sistemon (ekz. En ASCII-a datumbazo).

Although the x-system is now very popular everywhere, the Akademio recommends that it is only used when you can't use the h-system (e.g. in an ASCII database).

Esperanta Litero X-Sistemo
ĉ cx
ĝ gx
ĥ hx
ĵ jx
ŝ sx
ŭ ux
