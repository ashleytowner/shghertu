Esperantaj Klavaroj

Ĉi tiu retpaĝo prezentas programojn kaj instrukciojn por tajpi esperantajn literojn per via komputilo aŭ poŝtelefono.

This page presents programs and instructions for typing esperanto letters on your computer or phone.

Operaciumoj

  • Windows
  • macOS
  • Linux
  • Android
  • iOS / iPad OS

Windows

  • Tajpi: Programo tuj kiu konvertas vian enigon («sx», «gx», k.t.p.) al esperanta literojn.
    The "ABC - Extended" keyboard layout can type Esperanto letters. This table presents the keys you must use for Esperanto letters.
  • Tajlorita klavararanĝo de CodeWeaver: Kiam ĝi estas instalita, aldonu ĝin kiel kroma klavaro en la lingvagordoj.
    The "ABC - Extended" keyboard layout can type Esperanto letters. This table presents the keys you must use for Esperanto letters.

macOS

  • La klavararanĝo "ABC - Extended" povas tajpi esperantan literojn. Ĉi tiu etato prezentas la klavojn kiu vi devas uzi por esperantaj literoj.
    The "ABC - Extended" keyboard layout can type Esperanto letters. This table presents the keys you must use for Esperanto letters.
    Tajpu Por Tajpi
    ⌥+6 C/⌥+6 ⇧C ĉ/Ĉ
    ⌥+6 G/⌥+6 ⇧G ĝ/Ĝ
    ⌥+6 H/⌥+6 ⇧H ĥ/Ĥ
    ⌥+6 J/⌥+6 ⇧J ĵ/Ĵ
    ⌥+6 S/⌥+6 ⇧S ŝ/Ŝ
    ⌥+B U/⌥+B ⇧U ŭ/Ŭ
  • Tajlorita klavararanĝo de CodeWeaver
    Custom-made keyboard layout by CodeWeaver
  • macOS-a kopio de Tajpi
    A macOS clone of Tajpi

Linux

Plej multaj Linux-a distribuoj jam havas esperantan klavaron. Vi povas trovi ĝin en la agordoj.

Most linux distributions already have an esperanto keyboard. You can find it in the settings.

Android

Vi devos elŝuti aparte liveritan klavaron, kiu subtenas Esperanton. En Android vi havas du eblojn:

You will need to download a third-party keyboard that supports Esperanto. On Android you have two options:

  • Gboard
  • Swiftkey

iOS / iPad OS

Vi devos elŝuti aparte liveritan klavaron, kiu subtenas Esperanton. En iOS, SwiftKey estas la plej bona eblo.

You will need to download a third-party keyboard that supports Esperanto. On iOS, SwiftKey is the best option.

HTML

Por tajpi esperantajn literojn en HTML, oni devas uzi HTML-ajn estaĵojn. Jen etato kiu prezentas esperantajn literojn kaj ilian respondan HTML-an estaĵon.

to type esperanto characters in HTML, you must use HTML entities. Here is a table which shows esperanto letters and their corresponding HTML entity.

Esperanta Litero HTML-a estaĵo
Ĉ &Ccirc;, &#x108;, &#264;
ĉ &ccirc;, &#x109;, &#265;
Ĝ &Gcirc;, &#x11C;, &#284;
ĝ &gcirc;, &#x11D;, &#285;
Ĥ &Hcirc;, &#x124;, &#292;
ĥ &hcirc;, &#x125;, &#293;
Ĵ &Jcirc;, &#x134;, &#308;
ĵ &jcirc;, &#x135;, &#309;
Ŝ &Scirc;, &#x15C;, &#348;
ŝ &scirc;, &#x15D;, &#349;
Ŭ &Ubreve;, &#x16C;, &#364;
ŭ &ubreve;, &#x16D;, &#365;

Aliaj Sistemoj por Skribi

Kiam oni ne povas tajpi esperantajn literojn, ekzistas du aliajn sistemojn por prezenti ilin: La h-sistemo kaj la x-sistemo.

H-sistemo

La Akadaemio rekomendas, ke por aferoj, kiujn oni intencis legigi, oni uzu la h-sistemon.

Esperanta Litero H-Sistemo
ĉ ch
ĝ gh
ĥ hh
ĵ jh
ŝ sh
ŭ u

X-Sistemo

Kvankam la X-Sistemo nun estas tre populara ĉie, la Akademio rekomendas, ke oni nur uzas ĝin, kiam oni ne povas uzi la h-sistemon (ekz. En ASCII-a datumbazo).

Esperanta Litero X-Sistemo
ĉ cx
ĝ gx
ĥ hx
ĵ jx
ŝ sx
ŭ ux
